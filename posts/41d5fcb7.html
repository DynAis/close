如何使用VSCcode调试Arduino

12月 17, 2019 发布在 note, arduino

一.为VSC装入Arduino扩展

安装扩展前,需要确定Arduino IDE已经安装在电脑里

请选择与VSC匹配的版本,如当前为1.8.8

注意

第一次使用1.8.9的时候,因为版本太高,文字出现了乱码的情况

然后是在VSC中安装Arduino官方拓展
直接搜索即可
拓展包括如下功能

  • IntelliSense and syntax highlighting for Arduino sketches
  • Verify and upload your sketches in Visual Studio Code
  • Built-in board and library manager
  • Built-in example list
  • Built-in serial monitor
  • Snippets for sketches
  • Automatic Arduino project scaffolding
  • Command Palette (F1) integration of frequently used commands (e.g. Verify, Upload…)
  • Integrated Arduino Debugging New

做完这些以后记得在Arduino的setting.json配置文件中加入Arduino的文件夹位置
不然无法使用

二.快捷键

F1搜索Arduino也有很多相关的信息
